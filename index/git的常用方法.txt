一，Git 分支
	1，创建本地分支
		$ git branch 分支名称 
	2，查看各个分支当前所指的对象
		$ git log --oneline --decorate --all
	3，切换本地分支
		$ git checkout 分支名称
	4,新建分支并切换到新建分子上
		$ git checkout -b 分子名称
		$ git master 分支名称（注意，此时已切换master,将新分支合并到主分支上)
	5,查看所有分支
		$ git branch
	6,删除分支
		$ git branch -D 分支名称
二，分支管理
	1，查看每个分支最后一次提交
		$ git branch -v
	2, 合并或尚未合并到当前分支的分支
		$ git branch --merged
		$ git branch --no-merged
三，分支开发工作流
	1，长期分支，你可以用这种方法维护不同层次的稳定性。
	2，特性分支，特性分支对任何规模的项目都适用。
四，Git 分支 - 远程分支
	1，远程分支
		1，你可以通过 git ls-remote (remote) 来显式地获得远程引用的完整列表。或者通过 git remote show (remote) 获得远程分支的更多信息。
